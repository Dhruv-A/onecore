7 Functions

7.1 Functions

A function is a group of code that which can be run repeatedly (when called repeatedly). It allows you to define code once and reuse it many times (to avoid writing out the same code multiple times). You can pass functions data (known as parameters), and the function can return data.

For example, I can make a (trivial) function to do addition, which takes 2 parameters (a, b) and returns their sum:

int add(int a, int b) {
  int output = a + b;
  return output;
}

You can then use the function by calling it:

cout << add(2, 3) << "\n";
cout << add(-1, add(7, 8)) << "\n";

You can have a function without a return value by using void

void sayHi() {
	cout << "Sup world!";
}

More generally, a function will have the form: return_type function_name(parameter1, parameter2, ...) {

In fact, int main() is a function, which is called when code is executed. (The return value, known as the exit code, is by default 0, unless the program has an error)

It is essential you use functions. It makes debugging easier, since you can divide your code into functions and check that each function works. Also, it is cleaner to read (and when other people read your code).

7.2 Recursive Functions

"Why are recursive functions so narcissistic? Because they always call themselves."

Recursive functions allow a complex idea or problem to be defining by simpler versions of itself. They are functions which call themselves to solve a smaller subproblem of the original problem. Usually, there is a base case to stop the function from calling itself infinitely.
For example, "I think you think I love you" contains the sentence "You think I love you", contains the sentence "I love you".

Another example is the recursive definition of the factorial function:

int factorial(int n) {
	if (n == 1) {
		return 1; // base case
	}
	return n * factorial(n - 1); // recurrence relation
}

void printCombinations(int depth) {
	if (depth > 3) return; // termination condition
	for (int i = 1; i <= 3; i++) {
		// output i, padded with sufficient spaces
		for (int rep = 1; rep < depth; rep++) {
			cout << " ";
		}
		cout << i << "\n";
		
		// recurrence relation
		printCombinations(depth + 1);
	}
}
