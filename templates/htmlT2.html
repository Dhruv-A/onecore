2. Basic Layout

2.1 Tags

Tags are the building blocks of all websites written in HTML. Without them, the actual code couldn't be interpreted. Different tags are used for certain circumstances, in order to help the web developer give meaning to certain parts of their website. For example, the <p></p> tag, is used to wrap paragraphs of text, in order to help give a standardised design across the website. In fact, this text you're reading right now, is wrapped in a paragraph tag.

Some common tags include <p>, <html>, <body>, <head>, <h1> and <title>

2.2 Main Structure of a Website

Almost every website ever written in HTML have a few things in common. For starters, every single HTML document will start with the declaration <!DOCTYPE html> This is so the program interpreting the document, i.e. Chrome, knows that the site is written in HTML.

Every website will also have a <html> tag, which denotes the HTML portion of the website. Then, most well designed websited will include a <head> tag, which will normally wrap the <title> tag (the title tag wraps the text which will become the title of the page, which is the text in the tab of a browser), and any links to other pages, i.e. CSS Style sheets or javaScript pages.

Next, the websites actual content is usually wrapped in a <body> tag. In most cases, if your HTML document does not contain these tags, any good interpreter, will add them in automatically. Most websites will follow this general structure, an example of which is given below. 

<!DOCTYPE html>
   <html>
      <head>
         <title>Awesome Website</title>
         <link rel='stylesheet' href='style.css'>
      </head>
      <body>
         <h1>Heading of the page</h1>
         <p>Some information in a paragraph</p>
         <p>More information in a different paragraph</p>
      </body>
   </html>

2.3 How to view a HTML file locally

For starters, you will need to create a HTML file. If you are using any of the text editors mentioned in lesson 1, you will just need to create a file, within your project folder. Suitable tutorials for each of these programs can be found on their websites.

When creating a HTML file, the main things to remember, are getting the file extention right (.html), and following proper basic design principles. Go ahead and create a new HTML file, named index.html, and copy paste the code above into it, using your text editor. Then go back to your file explorer, and open the file with your prefered web browser (As mentioned before, Chrome is the prefered choice for this course, as it has certain shortcuts that will be utlisised later in the course). If you don't know how to open a file using your web browser, on windows, just right click your file, and click "open with", and then select your web browser.
