4. Basic Tags 2

4.1 Selecting Images

We will continue to use the bookshop example from the previous module and build on it below. Our webpage currently only has text and looks boring. One way to make a page more interesting is using images. You can add an image you find online using its url or anything you have on your laptop using its name.

One very important aspect of a website is the images you use. You cannot copy any image you find online and must look at any conditions for using it. For example, we make most of the images we use on this website and the remaining ones come from sources which allow images to be reproduced. We recommend you use a source such as unsplash which provides freely-usable images.

4.2 Adding Images to HTML

To add images onto our website, we use the image tag. It has certain attributes which define what is inside. The first is src which takes in the source of the image (either a url or the image name if it's saved in the same folder as the HTML file). The other is alt which specifies text that your webpage will display if it cannot find the image. Furthermore, image tags do not have a closing tag. Let's take a picture of books from unspash and put it on the page.

<img src="https://images.unsplash.com/photo-1457369804613-52c61a468e7d?ixlib=rb-1.2.1&auto=format&fit=crop&w=1050&q=80" alt="books-image">

In the above example, it retrieves the image as expected. However, in some cases it cannot find the image, so it will use the specified alternative text.

<img src="https://images.unsplash.com/photo-14557369804613-52c61a468e7d?ixlib=rb-1.2.1&auto=format&fit=crop&w=1050&q=80" alt="books-image">

4.3 Hypertext Links

Sometimes we want to provide links to different websites and pages such as unsplash and atom. These can be created using the anchor tag. It requires a closing tag. Inside the opening tag it has the href attribute. This specifies the url of the page you want to link. In between the opening and closing tags, you specify what you want to be displayed on the link. Let's create a link to the home page of this website, which has the url https://onecore.tech/.

<a href="https://onecore.tech/">CORE website</a>

As you can see, the link doesn't look great and the hyperlinks on this website look way more appealing. By default, links are dark blue and underlined. However, we can change almost everything about the styling. We'll learn about that later on when we move to CSS.

4.4 Bold and Italics

Within paragraphs, we sometimes want to put emphasis on certain words. There are many rules about what to use and if you're interested in learning about it this is a good website. When we want bolded text to indicate words of greater importance, we use the strong element. This is what it looks like.

<p>Come to our bookshop to purchase books about programming!
   <strong>We are only open on weekdays.</strong></p>

Alternatively, we can use the emphasis element to indicate that a certain sentence has emphasis. This is what it looks like.

<p>Come to our bookshop to purchase books about programming!
  Our most popular book is <em>Learn HTML and CSS</em>.</p>

There are other methods to style text which are more preferred in some cases. We will start to cover styling in the next module!
