2.2 Files

To read and write to files, you need #include <fstream> at the top of your program (f stands for file).

Then you can open a file for reading using ifstream (Input File stream)

ifstream fin {"filenamein.txt"};

Or similarly open a file for writing using ofstream (Output File stream)

ofstream fout {"filenameout.txt"};


The following program reads a number from a file "in.txt", displays the number, and writes "Sup World" to a file "out.txt"

#include <iostream>
#include <fstream>
using namespace std;
int main() {
	ifstream fin{"in.txt"};
	ofstream fout{"out.txt"};

	// write some code
	int num;
	fin >> num;
	cout << "The file contains the number " << num;
	fout << "Sup world";
}

NOTE: It is not advised to read and write from the same file simultaneously.
