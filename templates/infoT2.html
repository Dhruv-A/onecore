2 ORAC

2.1 Using ORAC

The origins and meaning of "ORAC have remained unknown for many generations. Some say it is short for "Oracle". Some say it stands for "ORAC Recursive Acronym Competition". It will probably forever remain a mystery...

ORAC is a online training site for the Australian Informatics Olympiad programme, which allows you to view past problems. Register an account to get your solutions marked by an online evaluator.

We will be using ORAC to revise our C++ and informatics knowledge at the end of every lesson.

2.2 Standard Input/Output

Displaying a number is similar to displaying text in Sup World:

cout << 1;

You can also display a space:

cout << " ";

or display a new line:

cout << "\n";

or display multiple things on a single line:

cout << "Sup to " << "the world of " << "c++" << "\n";

To receive input, use cin:

int num; // this creates a variable num which is an int (integer)
cin >> num;
string your_name; // this creates a variable called you_name, stores some text
cin >> your_name;

2.3 File Input/Output

To read and write to files, you need #include <fstream> at the top of your program (f stands for file).

Then you can open a file for reading (called fin) which is an ifstream (Input File stream).

ifstream fin {"filenamein.txt"};

Or similarly open a file for writing (called fout) which is an ofstream (Output File stream).
ofstream fout {"filenameout.txt"};

Although you do not have to call your files "fin", "fout", you should choose a consistent and logical name.

The following program reads a number from a file "in.txt", displays the number, and writes "Sup World" to a file "out.txt".

#include <iostream>
#include <fstream>
using namespace std;
int main() {
	ifstream fin {"in.txt"};
	ofstream fout {"out.txt"};

	// write some code
	int num;
	fin >> num;
	cout << "in.txt contains the number " << num;
	fout << "Sup world";
}
