4. Decisions

4.1 If statement

If statements compare multiple value and output true or false. The syntax for an if statement is as follows:

if (condition)
{
  // block of code to be executed if the condition is True
}

Comparison Operators:

  • Less than: a < b
  • Less than or equal to: a <= b
  • Greater than: a > b
  • Greater than or equal to: a >= b
  • Equal to a == b
  • Not Equal to: a != b

Logical operators:

Indentation is not significant in C#, because when the code is being compiled, C# converts everything into one line, and determines new statements by the semi colons (this is why we need to include semi colons at the end of each line.
