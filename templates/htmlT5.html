5. Styling 1

5.1 Inline, Internal and External Styling

There are many different ways to style HTML content using CSS.

  • Inline Styling is the first way to add styling. It is not very efficient and hard to read but is quick and applies to a single element. It works by adding the style attribute to a HTML tag.
  • Internal Styling works by adding all the styling at the top of a HTML file before the content that will be displayed. This means we can put all our code in one file, however, it can make the file really long.
  • External Styling is preferred for most websites, especially bigger ones with multiple webpages. In this, all the styling is put in a separate file which we link to the HTML file. This is the method used to style this website and is the most common method you'll find.

5.2 Colour

The first useful thing we will learn is changing the colour of text. This can be used to change the colour of text in headings, paragraphs, hyperlinks etc. When we program in HTML, we use the American preferred spelling for words, so to change the colour, we use the color property. Because we will implement this inline, we will put the styling in the style attribute and surround the styling with quotation marks.

<p style="color: green;">CORE Bookshop</p>

There are many different colours you can specify by name. However, you can also use the rgb value of a colour, or its hex value.

<p style="color: rgb(50, 168, 82);">CORE Bookshop</p>

Let's break down the above examples. When we use CSS styling, we always need to end each line with a semi-colon. This is so that the code knows that is the end of a line. Additionally, after the property name, such as color, we put in a colon before the value.

5.3 Font Size

We can change the font size using the font-size property. The value specified can range in terms of form. Let's look at each type one by one. Firstly, we can specify words such as small and large.

<p style="font-size: small;">Lorem ipsum dolor sit amet</p>
<p style="font-size: medium;">Lorem ipsum dolor sit amet</p>
<p style="font-size: large;">Lorem ipsum dolor sit amet</p>

Another way to specify size is using pixels, which are a set unit of measurement. This is denoted with px as shown below.

<p style="font-size: 10px;">Lorem ipsum dolor sit amet</p>
<p style="font-size: 15px;">Lorem ipsum dolor sit amet</p>
<p style="font-size: 20px;">Lorem ipsum dolor sit amet</p>

The final way we will cover here is using a percent proportion to the parent element. Basically, this is a relative size and is good for adjusting to different screen sizes.

<p style="font-size: 50%;">Lorem ipsum dolor sit amet</p>
<p style="font-size: 70%;">Lorem ipsum dolor sit amet</p>
<p style="font-size: 90%;">Lorem ipsum dolor sit amet</p>

5.4 Font Family

Changing the font style of the text is very common and important for the style of our webpage. You can choose from various provided font types and generic names. Some font names you can use include Times and Arial and some generic names include serif and cursive. It is common to provide fallback fonts when we specify a font in case the desired font cannot be found. Let's say I want a paragraph to have Times New Roman as the font.

<p style="font-family: 'Times New Roman', Times, serif;">Lorem ipsum dolor sit amet</p>

There are a few things to consider. What the styling above does is try to make the text Times New Roman but if the browser doesn't support that, it tries Times which is less specific and more common. If this also fails, it defaults to serif and lets the browser choose an available serif font. These extra fonts we specify as backup are called fallback fonts. Note that when you specify the name of an actual font like Times New Roman and Arial, you have to put them in quotation marks.
