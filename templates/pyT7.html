7. Loops

7.1 While Loop

While loops allow you to repeat a block of code as long as a certain condition is met. It works by checking if the condition next to while is true. The most basic while loop is the true loop in which your block of code will run for an infinite time:

while True:
  print('I want soup')

You can try this in the terminal but be sure to press ctrl+c to quit the code or else it will run forever. These loops can be useful in situations where we just want a block of code to run indefinitely e.g. in a code that constantly checks the temperature of the soup. However, more commonly we use a pre-test while loop. For example we can make a code to print “keep cooking” until the time of cooking reaches 10 seconds.

soup_time = 1
while soup_time < 10:
  print("Keep cooking")
  soup_time += 1

We define a variable outside the loop and set it to be 1. Let’s say that each iteration is equal to one second. This code should print “keep cooking” 10 times.

soup_time = 1
while soup_time < 10:
  print(f'The soup_time is {soup_time}')
  print('Keep cooking')
  soup_time += 1
print('We are out of the loop')

7.2 For Loop

We can consider for loops as a sort of while loop as they allow you to perform a section of code multiple times. In our example, let’s do the same thing with a for loop. 

for i in range(1, 10):
  print('Keep cooking')

This looks a bit different to a while loop. The first thing different is i, which is a variable that is used to store the integer value of the current position in the range of the loop (it might seem complicated right now so don't worry if you don't completely understand it).

Range is an in built function that creates a sequence of numbers from 1 to 10. (1, 2, 3, 4, 5, 6, 7, 8, 9). One thing to note about the range function is that the last number it gives you is the final number minus 1. In the first iteration i is assigned to 1, then 2 on the second time through the loop etc. We can see this if we print out i. 

for i in range(1,10):
  print(i)
