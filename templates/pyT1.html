1. Introduction

1.1 Basics

Hi, I’m Dhruv and I created CORE as a way to provide students with programming opportunities! CORE currently has several programs and courses aimed at and run by high school students in areas like game making, web development and 3D-Modelling.

All of the tutorials and articles on this website are free to use!. Additionally, we have a discord server for everyone, whether you have programming queries, want to start a project with others, write your own articles for our website or just want to provide suggestions for future content!

1.2 Introduction to python

Python is very readable programming language. It has simpler syntax than many other programming languages and there is a lot of emphasis on readability and simplicity. In recent years, python has grown in popularity very drastically and is now one of the most widely-sought languages.

1.3 Downloading python

You can download python from python.org. Make sure to download python 3, of which the latest version is currently python 3.8.

Alternatively, you can download it from the Microsoft store if you're on windows.

After downloading it, you will get a few different applications such as Spyder and IDLE. If this is your first time programming, I recommend using IDLE because of its simplicity.
