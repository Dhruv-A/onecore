4. Arithmetic Operators

4.1 Basics

As a high-level programming language, python can be used to do math calculations.

Let's first step through how to use the first four operators in the interpreter.

>>> 2 + 5  # Addition
7
>>> 8 - 3  # Subtraction
5
>>> 3 * 4  # Mutiplication
12
>>> 10 / 2 # Division
5.0

Notice that 10 / 2 returns 5.0 with an extra .0 instead of just 5. Up until now we have only be dealing with numbers as integers which are whole numbers like:

  • 5
  • 0
  • 256
  • -12

However there is in fact another number-like data type called floats which are numbers with decimal places like:

  • 1.5
  • 5.0
  • 3.1415926

Floats are necessary when we divide a number by a non-factor:

>>> 7 / 2
3.5
>>> 4.5 / 3
1.5
>>> 10 / 3
3.3333333333333335

4.2 The floor division operator

The floor division operator performs ordinary division, then rounds the result down to the lower integer.

>>> 10 / 3
3.3333333333333335
>>> 10 // 3
3
>>> 8.8 / 4
2.2
>>> 8.8 // 4
2
>>> 7 / 2
3.5
>>> 7 // 2
3

Notice that the final result is not the nearest integer, but rather the lower integer.
If we want to divide integer and return an integer instead of a float we can use the foor operator.

>>> 10 / 2
5.0
>>> 10 // 2
5

4.3 The modulo operator

The modulo operator, sometimes known as the remainder operator returns the remainder when the first operand is divided by the second.

>>> 5 % 3
2

In the above example, 3 goes into 5 once, leaving a remainder of 2. Therefore, the result of 5 % 3 is the remainder 2.

>>> 23 % 5
3

In the above example, 5 goes into 23 four times, leaving a remainder of 3. Hence the result of 23 % 5 is the remainder 3

4.4 The exponent operator

The exponent operator or power operator returns the first operand raised to the power of the second.

>>> 5**2  # Equivalent to 5 * 5
>>> 2**3  # Equivalent to 2 * 2 * 2

4.5 Order of operations

If you use multiple arithmetic operators on a single line, Python will use the conventional order of operations. In decreasing order of precedence:

  1. *, /, //, %
  2. +, -

Operators on the same line above are performed in order of occurence.

>>> 3 * 2 + 5  # Equivalent to 6 + 5
>>> 4 * 5 - 3 * 4  # Equivalent to 20 - 12
8
>>> 7 / 2 * 3  # Equivalent to 3.5 * 3
10.5
>>> 3 + 10 // 2  # Equivalent to 3 + 5
8
>>> 3 * -5
-15

Brackets can be used to specify order of operations.

>>> 3 * (2 + 4)  # Equivalent to 3 * 6
18
>>> 12 / (1 + 2)  # Equivalent to 12 / 3
4
>>> (5 + 9) % 3  # Equivalent to 14 % 3
2

4.6 Arithmetic operators on variables

We can assign integers and floats to variables and perform arithmetic operations on these variables for the same result.

>>> a = 12
>>> b = 3
>>> c = 1.5
>>> a + b
15
>>> a - b
9
>>> a * b
36
>>> a / c
8.0
>>> a // b
4
>>> c ** b
3.375

4.7 Converting strings to integers

Sometimes, we want to convert a string to an integer so that we can perform the above maths on it. This can be done using int().

>>> num = 5
>>> print(type(num))
<class 'int'>
>>> num = '5'
>>> print(type(num))
<class 'str'>
>>> print(type(int(num)))
<class 'int'>

When we want to get an input from the user, we use input(). However, sometimes we want to get an an integer input from the user. The input() function gives you a string, so we can simply surround it in int().

>>> year = input('What year is it? ')
What year is it? 2020
>>> print(type(year))
<class 'str'>
>>> year = int(input('What year is it? '))
What year is it? 2020
>>> print(type(year))
<class 'int'>

As you can see, when we don't convert our input to an integer, even if we enter a number, it is a string. But when we convert it to an integer using int(), it makes it an integer!

4.8 More operators

Sometimes it's important to compare two values. We haven't come across any examples of this so far but we will very soon!. To compare values we have a few operators which may seem familiar from maths.

  • ==  checks whether two values are equal
  • !=   checks whether two values are not equal
  • >    checks whether the value of the left operand is greater than the right operand
  • <    checks whether the value of the left operand is less than the right operand
  • >=  checks whether the value of the left operand is greater than or equal to the value of the right operand
  • <=  checks whether the value of the left operand is less than or equal to the value of the right operand

Here are some examples of these being used: 

>>> print(5 == 5)
True
>>> print(5 != 5)
False
>>> print(5 <= 10)
True
>>> print(5 <= 5)
True
>>> print(5 <= 4)
False

These are how we compare values and are very important so make sure to remember them!
