3. Basic CSS

3.1 Linking Stylesheets

It is very common to find, on any professional website, that in the head of the website HTML, a link element will connect a "stylesheet" to the page. this will normally look like <link rel='stylesheet' href='main.css'>. In this example, inside of the link tag, 2 other attributes are set, these attributes are rel and href. The rel attribute defines the relation that the linked file has to the website. In this case, the linked file, main.css, is a stylesheet. And the href attribute gives the location of the file relative to the opened html page. If you're on Chrome, and press Ctrl+Shift+I, and select Elements, you can see the HTML code of this website, and see the link element in action.

3.2 What are stylesheets, and what's CSS?

Stylesheets, are files that contain CSS code, that can be quickly linked to each website's individual pages, in order to reduce file size and increase the readability of a website's code.

CSS is a language seperate to HTML, used to style websites. An example of CSS code, is shown below. 

body {
   background-color: red;
   font-family: monospace, serif;
   color: red;
   font-size: 16px;
}

In the above example, the style of the body element is being edited.

In the above CSS, the styling is setting the background image of the website to 'red' (remember the body element is the element in which the entire website is placed).

The font-family is then set to monospace, with serif as a backup. This set's the font of all the text in the body element. The reason 2 fonts are supplied, is if a browser or computer is incapable of rendering the first font, they can have a backup before defaulting to a standard font.

The color attribute, defines the color of the text within the body element, the color attribute accepts HTML colors, (i.e. red, blue, skyblue, rebeccapurple), but also accepts hex codes, and rgb codes. To input hex code, you must place a # before the hex code. To input rgb or rgba code, you must format it as rgb(r, g, b) or rgba(r, g, b, a), with the r, g, b representing the 8bit color values [0-255] of each color, red, green and blue. The a in rgba represents the opacity of the color, and is a float value between 0 and 1.

The font-size attribute, sets the size of the text in the element. This is comparable to setting the font size of a paragraph of text in a word-processing program.

3.3.1 CSS Syntax explained

Starting off, in the CSS example in 3.2 the body element, is the selector. It points the the element in the HTML which you want to style.
Inside of the curly brackets are a series of declarations. A declaration is made up of a property, and a value. For instance the declaration "color: red;" is made up of the property: "color" being set to the value "red".

3.3.2 CSS Selector syntax

As explained in 3.3.1, the selector points to the HTML element being styled, however there are many ways a selector can be defined, in order to style specific situations.

Class Styling: When you want to style a certain class, all you need to do is place a . before the class name.
i.e. the following CSS hides all elements with the class 'hidden'. 

.hidden {
   display: none;
}

Element specific Class Styling: When you only want to style a specific element that has a Class attached to it, when you write the Selector, all you need to do is follow the format : "ELEMENT.CLASS". This is so you can seperately edit multiple elements that contain the same class.
i.e. the following CSS sets the text-color all paragraph elements with the class 'red' to red, but sets the background-color of all div elements with the class 'red', to red. 

p.red {
   color: red;
}

div.red {
   background-color: red;
}

ID Styling: When you only want to style a specific ID, all you need to do is place a # before the name of the ID.
i.e the following CSS sets the margin all elements with the ID 'dogPhoto1' to auto, which horizontally centers an element. 

#dogPhoto1 {
   margin: auto;
}

Universal Styling: When you want to apply the same style to every element, just make the selector the Universal selector, *
i.e. the following CSS sets the text-color of every element to red. 

* {
   color: red;
}
