3. Basic Tags 1

3.1 DOCTYPE Declaration

The HTML which makes up almost every website you'll find is called HTML5. When the web browser interprets what our HTML says, it needs a way to know that we are using HTML5. Therefore, we start off every HTML page we code with the doctype declaration below.

<!DOCTYPE html>

3.2 Headings

The first tag we will learn is the heading tag. As the name suggests, you use them for headings. Let's say you're making a website for a bookshop, the first thing you want on the page will be a heading so that the viewer knows what the page is about.

<!DOCTYPE html>
<h1>Core's Bookshop</h1>

Let's break down the above code:

  1. The first line is the doctype declaration we explained above. You always include it as the first line in your file.
  2. The second line contains the heading using the <h1> tag. We need both an opening tag and the closing tag to signify the end of the tag. The closing tag is the same as the opening one except it starts with </

Feel free to copy code on this website and paste it directly into your HTML file using the copy icon at the top right corner of the blocks of code!

Let's see what the above code looks like by opening it in chrome.

There are many other heading tags we can use. These include <h2>, <h3>, <h4> and so on.

<!DOCTYPE html>
<h1>H1 Heading</h1>
<h2>H2 Heading</h2>
<h3>H3 Heading</h3>
<h4>H4 Heading</h4>

3.3 Paragraphs

One really common and important tag is the paragraph tag. This tag is used to denote paragraphs of text. Let's say that our bookshop website has some text about when and what we will sell.

<!DOCTYPE html>
<h1>Core's Bookshop</h1>
<p>Welcome to our bookshop! We're opening next month and
   will be selling books on all types of programming.</p>

Notice how even though I put the content in the paragraph tag on two separate lines, it displayed with all the content on the same line. This is a feature of HTML, where the file ignores new lines in the code and treats them as if they were one space long. This is the case for everything in HTML, not only paragraphs.

3.4 New lines in HTML

When we want to display content on separate lines, we use the break tag. This does not require a closing tag and instead inserts a line break where you place it. Let's apply it to our previous example to split the paragraph into two separate lines.

<!DOCTYPE html>
<h1>Core's Bookshop</h1>
<p>Welcome to our bookshop!<br>
   We're opening next month and
   will be selling books on all types of programming.</p>
