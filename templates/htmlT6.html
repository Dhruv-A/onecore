6.1 Getting online fonts

You might've noticed that the font used in this course isn't any of the fonts we've covered. We selected this font because it fits the design of this website and is easy to read. You can find fonts online and add them to your website. One really easy way to get fonts from online is using Google Fonts. The process is really simple. Firstly, find the font you want. We're going to choose Roboto for this example. Click on it and select the style you want. Once you click Select this style, a sidebar will pop up on the right. Go to the tab with Embed.
Now, there are two options:

  1. <link>
  2. @import
Either option works fine! The options for Roboto are

<link href="https://fonts.googleapis.com/css2?family=Roboto&display=swap" rel="stylesheet">

and

@import url('https://fonts.googleapis.com/css2?family=Roboto&display=swap');

Note that the second option needs to be put in your CSS file (we'll cover this later) or in style tags (we'll cover this when we move on to internal styling). After you import a font, you can use it inside the font-family property. Make sure to surround the font with quotation marks!

<p style="font-family: 'Roboto', Helvetica, sans-serif;">Lorem ipsum dolor sit amet</p>

6.2 Text Align

Sometimes, we want our text to be at the right side or centre. This can be specified using the text-align property. It can take in many different values:

  • left is the default value which aligns the text to the left
  • center aligns the text to the centre
  • right aligns the text to the right
  • justify adjusts the text so that each line has the same width
<p style="text-align: left; width: 50%;">Lorem ipsum dolor sit amet, consectetur adipiscing elit, sed do eiusmod tempor incididunt ut labore et dolore magna aliqua. Ut enim ad minim veniam, quis nostrud exercitation ullamco laboris nisi ut aliquip ex ea commodo consequat.</p>

6.3 Width

As you might've noticed above, I also specified width: 50% in the previous examples. This was to make it easier to notice different values of text-align. The width property is very versatile and is used to specify the width of an element. Like with font size from before, it can take in a range of values. The most common one we will look at is percentage values. Percentages are relative values and depend on the original size of an element.

6.4 Text Decoration

Text decoration defines certain decorative properties of text. This includes underlined and strikethrough text. Remember the default appearance of hyperlinks was not very pleasing. We can change that using the text-decoration property. Below is a normal link:

<a href="https://onecore.tech/">CORE Website</a>

We can set the text-decoration property to none to remove the underline from the link.

<a style="text-decoration: none;" href="https://onecore.tech/">CORE Website</a>

Additionally, we can change the colour of the link to something that matches the website better.

<a style="text-decoration: none; color: rgb(66, 138, 245);" href="https://onecore.tech/">CORE Website</a>

Conversely, we can underline text to make it stand out. Let's take the bookshop example from before and underline the heading to make it stand out.

<p style="text-decoration: underline blue ;">CORE's Bookshop</p>

Within the property, we can specify multiple conditions.

<p style="text-decoration: underline blue wavy;">CORE's Bookshop</p>
<p style="text-decoration: underline red dotted;">CORE's Bookshop</p>

There are other things you can do with text-decoration.

<p style="text-decoration: overline blue;">CORE's Bookshop</p>

6.6 Internal Styling

Styling within elements is sometimes really convenient, however, it can make the HTML really hard to read and is all over the place. Another way to style our HTML is using internal styling. We will discuss the layout for this in the next module, however, I'll go through the basics here.

Firstly, you need to use style tags at top of the page to specify that you will put styling there.

<style>

</style>


<p>This is a paragraph</p>

Let's say that we want all paragraphs to be coloured blue. We can specify this inside the style tags. We first specify the element we want to style and then use curly braces {} to put the styling in.

<style>
p {
  color: red;
}
</style>


<p>This is a paragraph</p>

Every single paragraph will be coloured red because of this styling. If we want all paragraphs to also be 50% width and all h1 headings to be blue at the same time, we can also specify those as well.

<style>
p {
  color: red;
  width: 50%;
}
h1 {
  color: blue;
}
</style>

6.7 Background Colour

The background colour of the page can be set using background-color. When we want the whole page to be a specific colour, we can set the background colour of the body to that colour.

<style>
body {
  background-color: black;
}
p {
  color: white;
}
</style>
