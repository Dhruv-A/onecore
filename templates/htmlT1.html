1. Introduction

1.1 Introduction to HTML

HTML (Hyper Text Markup Language) is currently the industry standard for building websites.

On it's own, it is only capable of building mostly static websites, however, when combined with other languages, such as CSS, javaScript and PHP, much more complex and interactive websites can be built

In this course, we will be teaching basic HTML and CSS so that you can build websites yourself

1.2 Before you get started

Before you can get started with this course, you will need a few things, some of which, you hopefully already have.

You will need:

  • A fast Web Browser (Preferably Chrome or Chromium based)
  • A Text Editor (Programs such as Atom, Sublime Text or Visual Studio are suggested)
  • Willingness to learn
  • A Method for storing notes (For most people, writing notes can help with the retention of learnt information)

1.3 Course goals

By the end of this course, we hope to equip you with the skills required to read and write HTML and CSS and gain an understanding of basic web design concepts. If the content is getting too difficult, make sure to look back on previous content and revise our articles regularly.

I hope you have a great time learning HTML!
