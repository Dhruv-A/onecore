6 Arrays

6.1 Single Dimensional Arrays

Arrays are used to hold a collection of items. The data type must be the same for all elements. They can only hold 1 data type (e.g. int), and the size of the array is fixed. It is a good idea to declare arrays only directly after using namespace std;. There are 2 ways to declare arrays:

(most commonly used) Array declaration by specifying a constant size, elements are by default initialised to 0.

int foo[5];

Array declaration by specifying elements

int foo [5] = { 16, 2, 77, 40, 12071 };

Arrays are 0 indexed, which means foo[0] is 16, foo[1] is 2, and so on...


NOTE: if the problem statement requires you to have an array of size 100000, you should always initialise it to a slightly higher value (e.g. 100005).

6.2 Multidimensional Arrays

Multidimensional arrays are "arrays of arrays". For example, a 2D array can be likened to a grid (all of some data type). For example, a tic tac toe board can be represented as int grid[3][3];

grid[1][1] = 1; // centre cell
grid[0][1] = 2; // top edge cell
grid[2][0] = grid[0][1]; // bottom right cell
cout << grid[2][0];

NOTE: arrays can have more than 2 dimensions. For example int year[365][24][60][60];
