3 Numbers

3.1 Declaration & initialisation

When declaring, you need to specify the variable type and the variable name.

int x;

When initialising, you assign a value to the variable using = (Note: this is different to the way it is used in maths)

x = 1;

or more succinctly

int x = 1;

Another way to initialise a variable is from user input:

cin >> x;

Note that you have to declare a variable before you initialise it.

3.2 Display (cout)

Displaying a number is similar to displaying text in Sup World

cout << 1

You can also display a space

cout << " ";

or display a new line

cout << "\n";

or display multiple things on a single line

cout << "Sup to " << x << " worlds" << "\n";

3.3 Number Variable Types

To store a number in programming, you typically use an int. If you an extra long number, use long long (hahaha funny joke). If you have decimal, use double or long double. All of these can do the same operations, so for brevity I will only demonstrate its usage using only int

Operators

x * y

multiplication

 
x / y

division
for int and long long, the output of division is rounded down to the nearest integer

 
x + y

addition

 
x++

increment x by 1 (i.e. x = x + 1)
see here if you’re interested in the differences between x++ and ++x

 
x += y

increment x by y (i.e. x = x + y)

 
x - y

subtraction

 
x--

decrement x by 1 (i.e. x = x - 1)
see here if you’re interested in the differences between x-- and --x

 
x -= y

decrement x by y (i.e. x = x + y)

 
x % y

modulus, take the remainder when x is divided by y (only applies to int and long long)

The order of operations is consistent with BODMAS, and is computed form left to right:

cout << 2 + 13 * 5 + 2 << "\n";
cout << (2 + (13 * 5) ) + 2 << "\n"
