3 Numbers

3.1 Declaration & initialisation

When declaring, you need to specify the variable type and the variable name.

int x;

When initialising, you assign a value to the variable using = (Note: this has a different meaning to the equals sign in used in maths):

x = 1;

Or more succinctly:

int x = 1;

Another way to initialise a variable is from user input:

cin >> x;

NOTE: you have to declare a variable before you initialise it.

NOTE: = is used when assigning a value to a variable. For example,
x = x + 2 means take the value of x + 2, and assign it to the variable x (increment x by 2).
It does not mean solve the equation "x = x + 2".

3.2 Number Variable Types

To store a number in programming, you typically use an int. If you an extra big number, use long long (inesrt funny joke here). In particular, an int can hold any integer between -2147483648 and 2147483647 inclusive (2×10⁹ is a good rule of thumb). A long long can hold any integer between -9,223,372,036,854,775,808 and 9,223,372,036,854,775,807 inclusive (9×10¹⁸ is a good rule of thumb).
If you have decimal, use long double.

Operators

x * y

Multiplication.
x / y

Division.
For int and long long, the output of division is rounded down to the nearest integer.
x + y

Addition
x++

Increment x by 1.
See here if you’re interested in the differences between x++ and ++x
x += y

Increment x by y.
x - y

Subtraction.
x--

Decrement x by 1.
x -= y

Decrement x by y.
x % y

Modulus - take the remainder when x is divided by y (only applies to int and long long).
For example, x % 2 will be 1 if x odd or 0 if x is even.

The order of operations is consistent with BODMAS, and is computed from left to right:

// these 2 statements are equivalent
cout << 2 + 13 * 5 + 2 << "\n";
cout << (2 + (13 * 5) ) + 2 << "\n";
