1. Introduction

1.1 Basics

Hello! Welcome to the C++ and informatics course. I’m Ryan, a programmer and content writer at CORE.

Informatics is an area of programming focused on solving algorithmic problems. Typically, you are provided with a problem (i.e. sort an array), and you have to figure out a fast and optimal way to solve the problem, and implement it in a chosen language.

In my opinion, C++ is a really useful language to know. Although it may not be used as much in industry, it has many advantages such as fast runtimes and high-level built-in functions which come in very handy for informatics. It is quite common for professional informatics programmers to use C++.

This course aims to teach C++ as in introduction to programming and informatics, and will hopefully encourage you to continue your journey in aspects of programming. As such, I may not cover certain aspects of the language which are not necessary for informatics

In each lesson, I will provide some tutorials on either C++ functionality or a new algorithm, followed by some exercises or problems on orac to solve. You are not required to solve all of the problems since some may be quite challenging. It is ok to read over each lesson several times or revisit previous lessons, since I will write them as succinctly as possible.

1.2 Sup World

#include <iostream>
using namespace std;

int main() {
	// This is a comment
	cout << "Sup world!";
}

Sup world is a very basic program, which displays the text "Sup world!"

Line 1 #include <iostream>
#include means it includes a module. You are able to use the functionality from this module. In this case, you are able to use the functionality in “iostream”, which consists of displaying text.

Line 2 using namespace std;
This means that you can say “cout” instead of “std::cout” (line 5). Basically means your code can be shorter and more readable
"std" is short for standard

Line 3
Whitespace is good. Don’t write code like this!

Line 4 int main() {
Notice that since it starts with a {, it needs to close with a } (line 7)
Sometimes, you might see the { on the line after int main(). There is no difference, and I may use them both interchangeably

Line 5 // This is a comment
Everything after // is a comment, and is usually highlighted a different colour

Line 6
cout << " This means display to screen. Don’t forget the ; at the end of the line
NOTE: As a general rule of thumb, there is a ; on every line which doesn’t end with a { and doesn’t start with a #

1.3 Installing C++

Windows & Mac users
I use Visual Studio 2019 Community, although cLion is good as well.
There are many other programs you can use, but it comes down to personal preference.

Linux users
Lucky you! C++ compiler should be already installed on your machine. After creating your program file, use g++ -o output my_code.cpp to compile and ./output to run. Note that you can use additional flags on compilation which you might find useful

Once you have installed C++, try to implement Sup World without referring back to it
