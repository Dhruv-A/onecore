    1.2 Sup World

    #include <iostream>
using namespace std;

int main() {
	// This is a comment
	cout << "Sup world!";
}

    Sup world is a very basic program, which displays the text "Sup world!"

    Line 1 #include <iostream>
    #include is a way of includes a library of functions so that you are able to use them in your code. In this case, you are able to use the functionality in "iostream", which allows you to display text. (iostream stands for Input/Output stream)

    Line 2 using namespace std;
    This means that you can say "cout" instead of "std::cout" (line 6). Basically means your code can be shorter and more readable ("std" is short for standard)

    Line 3
    Whitespace is good. Don’t write code like this!
    int a,b,c,d,e[1001];int main(){cin>>a;for(int i=0;i<a;i++){cin>>b>>c;e[b]++;e[c]++;}for(int i=0;i<1001;i++)d=max(d,e[i]);for(int i=0;i<1001;i++)if(e[i]==d)cout<<i<<"\n";}

    Line 4 int main() {
    Every program has an int main(), which contains the body of the program. Notice that since it starts with a {, it needs to close with a } (line 7)
    Sometimes, you might see the { on the line after int main(). There is no difference, and I may use them both interchangeably

    Line 5 // This is a comment
    Everything after // is a comment, and is usually highlighted a different colour (in this case, light grey)

    Line 6 cout << "Sup world!"
    This means display to screen. Don’t forget the ; at the end of the line
    NOTE: As a general rule of thumb, there is a ; on every line which doesn’t end with a { and doesn’t start with a #

    1.3 Installing C++

    Windows & Mac users
    I use Visual Studio 2019 Community, although cLion is good as well.
    There are many other programs you can use, but it comes down to personal preference.

    Linux users
    Lucky you! C++ compiler should be already installed on your machine. After creating your program file, use g++ -o output my_code.cpp to compile and ./output to run. Note that you can use additional flags on compilation which you might find useful. I use -Wall -Ofast
