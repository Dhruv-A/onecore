2. Basic Setup

2.1 What are HTML Tags

Tags are the building blocks of all websites written in HTML. Without them, the actual code couldn't be interpreted. Different tags are used for certain circumstances, in order to help the web developer give meaning to certain parts of their website. For example, the <p></p> tag, is used to wrap paragraphs of text, in order to help give a standardised design across the website. In fact, this text you're reading right now, is wrapped in a paragraph tag.

Some common tags include <p>, <html>, <body>, <head>, <h1> and <title>
We will be covering many of these throughout this course

2.2 Viewing a HTML file locally

For starters, you will need to create a HTML file. If you are using any of the text editors mentioned in lesson 1, you will just need to create a file, within your project folder. Suitable tutorials for each of these programs can be found on their websites.

When creating a HTML file, the main thing to remember is getting the file extention right. When you name a HTML file, you need to add .html to the end.

I am using atom so the following images of the above process will be specific to atom, however, it should be almost identical for any text editor (sublime text, visual studio etc). Firstly, go to your text editor and create a new file.

Next, save that file in your project folder (it can be anywhere you want). When you save the file, remember to add .html to the end.

Now you're ready to start learning HTML!

When you want to run the file, you simply have to open it with chrome.

Because the file currently doesn't have any HTML in it, it will open to a blank chrome tab. However, as soon as we start coding and save our code, it should update when you refresh the page.
