3. Basics

3.1 Shell and Files

So far we've been working in the shell where we can see what we're running as soon as we type it. Another way to code is to create a new file in idle (File > New File) and then run after you type up code (Run > Run Module). You should always code in a file, however, we will explain most things in the shell so that you can instantly see the results.

3.2 Assigning Variables

Firstly, we will create a variable. Variables can be used to store information. Let's create a variable called 'name' and we will assign a name to it:

name = 'Dhruv'

Notice how you do not have to put quotations around the variable because it's not a string. Now, if we run this, and see what the value of the variable is:

>>> name
Dhruv

It returns the name.

3.3 Print

The print function outputs text:

>>> print('Hello, World!')
Hello, World!

Make sure the text inside is a string.

>>> print(Hello)
Traceback (most recent call last):
  File "<pyshell#1>", line 1, in <module>
    print(Hello)
NameError: name 'Hello' is not defined

This error is a result of not putting quotation marks around Hello. When we don't put in quotation marks, the code thinks we are looking for the variable Hello rather than the string. Additionally, you can print variables if you want!

>>> print('Hello')
Hello

Alternatively, you can store the name in a variable and print that out

>>>name = 'Dhruv'
>>> print(name)
Dhruv

3.4 Input

The input function asks for input from a user.

You can add text inside the function to ask questions. Then I can type my input on that same line, let's say I'm answering the question that I'm asking:

>>> input('What is your name?')
What is your name?Dhruv

Notice how the output text and your input is connected. This does not look good and we typically add a space at the end of the string to fix it:

>>> input('What is your name? ')
What is your name? Dhruv

When you ask for an input, you should store your answer in a variable so that you can use it later on:

>>> name = input('What is your name? ')
What is your name? Dhruv
>>> print(name)
Dhruv
>>> print('Hi ' + name)
Hi Dhruv

Notice that we don't use quotation marks around variables, and that's how our code knows that it's a variable and not a string!

3.5 Integers and Floats

An integer is a whole number. In python, if we want to assign an integer to a variable, we can do the same thing we did with strings, except without quotation marks. For example, if I assign the number 10 to num:

>>> num = 10
>>> print(num)
10
>>> type(num)
<class 'int'>

If we want decimals as well, we use something called a floating point number (float). For example, if I assign the number 10.1 to num:

>>> num = 10.1
>>> print(num)
10.1
<class 'float'>

So, those are integers and floats. We will look at reasons why integers and floats are important later!

3.6 Type casting

Sometimes, we want to convert between integers and strings. We can only concatenate strings, therefore, in order to join a string and an integer, we need to convert the integer into a string first. Let's say we want to print the temperature:

>>> temp = 20
print('It is ' + temp + ' degrees today!')
Traceback (most recent call last):
  File "<pyshell#3>", line 1, in <module>
    print('It is ' + temp + ' degrees today!')
TypeError: can only concatenate str (not "int") to str

As explained, you need will get an error if you try to concatenate a string and an integer. To fix this we change the integer to a string:

>>> temp = 20
>>> print('It is ' + str(temp) + ' degrees today!')
It is 20 degrees today!

3.7 f-Strings

Python has another way to connect multiple strings without using + to concatenate them. This is done using f-strings. A f-string is like a normal string, except if you want to add variables you add them as if they were part of the string, and surround them with {}. For example, let's say that we wanted to ask the user for a name and then greet them, we could do it in two ways. The first is:

name = input('What is your name? ')
print('Hi, ' + name)

This is the way we've already learnt: using + to concatenate two strings. The way below uses an f-string to do that:

name = input('What is your name? ')
print(f'Hi, {name}')

F-strings are much more readable and mean that we don't need to deal with multiple strings. Because of this, these modules will mostly be using f-strings.
