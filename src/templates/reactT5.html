5. Prop System

5.1 What is the prop system

A system for passing data from a parent component to its children components. Its ultimate goal is to customise or configure a child component. e.g for each of the comment children, it can display different data for each child component which is unique but ultimately the same changes.

5.2 How does it work

<CommentDetail [name of prop] = "[value of prop]" />

This will give a unique value for each property of a child component. Now in the component we have to run the function as prop. For example: 

const CommentDetail = props => {
  [also {props.author}]
}
Essentially the props is a built-in javascript object which contains all the information that is passed from the parent.

Another way to communicate that information is using closing tags 

<ApprovalCard>[whatever inside here becomes a prop]</ApprovalCard>
