14. Functions

14.1 In-built functions

Python has many in-built functiions, which are ones that python provides you. These include:

  • print()
  • enumerate()
  • min()
  • sorted()
  • chr()

I'll go through a few of these below

14.2 Enumerate

Enumerate is extremely useful in certain situations. It's a way to keep count of how far you are into a list when you're looping through it. For every value in a list that you iterate through in a for loop, it will have a tuple in the form of (count, value). Here's an example with and without enumerate.

>>> letters = ['a', 'b', 'c']
>>> for item in letters:
      print(item)

a
b
c
>>> for count, item in enumerate(letters):
      print(f'{count} {item}')
0 a
1 b
2 c

14.3 Min() and Sorted()

The min() function returns the lowest value. It's very straightforward.

>>> numbers = [6, 1, 3]
>>> print(min(numbers))
1

The sorted() function returns a list with sorted letters and numbers. You can sort lots of datatypes including strings and lists, but they will return lists.

>>> letters = 'helloworld'
>>> print(sorted(letters))
['d', 'e', 'h', 'l', 'l', 'l', 'o', 'o', 'r', 'w']
>>> numbers = '234094832842'
>>> print(sorted(numbers))
['0', '2', '2', '2', '3', '3', '4', '4', '4', '8', '8', '9']

14.4 Defining your own functions

Just like the in-built functions that python already has, we can define our own functions. Let's say we want a function that prints out 'Hello World' whenever we call it. We can do that as follows:

def hi():  # creates a function called hi
  print('Hello World')  # prints out 'Hello World' when the function is called

That's all you need! So, whenever we call the above function, it will print out 'Hello World'

def hi():
  print('Hello World')

hi()
hi()

This will print out that string twice:

Hello World
Hello World

Basically, how it works is, whenever you define a function and call it, it runs all the code in the function. It's great when you want to reuse certain code multiple times.

14.5 Take arguments

Just like most in-built functions in python, your own functions can take arguments. Let's say we wanted to make a function to greet someone. It takes in their name and prints 'Hi, ' followed by their name. We can make that function as follows:

>>> def sayHi(name):
      print('Hi, ' + name)
>>> sayHi('Harry')
Hi, Harry

You can take in multiple arguments into functions, separated by commas. Let's say you wanted a function that takes in two values and prints out their sum.

>>> def add(a, b):
      print(a + b)
>>> add(3, 4)
7

14.6 The return statement

Rather than printing out at the end, most functions return a value, which you can assign to another variable. We can create functions that do this using the return statement. They work like the previous functions, however, they will return a value rather than print it.

Let's say that we wanted to create a function that doubles an integer value.

>>> def double(a):
      b = 2 * a
      return b
>>> num = 5
>>> newNum = double(num)
>>> print(newNum)
10

This way, we can assign the output to a variable rather than only printing it out.
